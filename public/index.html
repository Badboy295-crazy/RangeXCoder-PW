Netlify Host Guide

How to host on Netlify

1
Login to Netlify: Go to netlify.com and sign in using your GitHub account.
2
Import from Git: Click "Add new site" -> "Import an existing project", select GitHub, and grant access to the RangeXCoder-PW repository.
3
Set Base Directory: Make sure the Base directory is set to empty (root) or build command is empty, and set the publish directory to public.
4
Redirect rules (Crucial): Since Netlify is hosting static files, any proxy requests under /api will need a backend server or a rewrite rule in a _redirects file.

Note: Static frontend hosting handles pages like index.html, batches.html, subjects.html etc. smoothly. If backend token-fetching/DRM endpoints are required, deploy the Node.js express backend (server.js) on Render or Koyeb and link it.
